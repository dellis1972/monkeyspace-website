# MonkeySpace Public Website

MonkeySpace is the official cross platform and open-source .NET conference.  Want to learn more about developing for the iPhone, Android, Mac, and \*nix platforms using .NET technologies?  How about developing games or learning more about open-source projects using .NET technologies?  MonkeySpace has provided an annual venue to collaborate, share, and socialize around these topics and more.

This repository is the public facing website for the MonkeySpace conference.
